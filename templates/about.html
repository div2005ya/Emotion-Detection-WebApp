About EmotionSense AI

EmotionSense AI is a web application that detects human emotions from facial images using Artificial Intelligence, Deep Learning, and Computer Vision.

🎯 Our Mission

To build an intelligent system that recognizes human emotions accurately and helps improve human-computer interaction.

🧠 Technologies

Python, Flask, OpenCV, DeepFace, TensorFlow, HTML, CSS, JavaScript, SQLite and SQLAlchemy.

🚀 Features

User Authentication, Image Upload, Emotion Detection, Webcam Detection, Dashboard and Prediction History.
